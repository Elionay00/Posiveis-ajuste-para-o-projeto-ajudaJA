HelpNow - Migração para React Native

Guia completo para transformar seu projeto web em um aplicativo mobile nativo

Passos para Migração

1

Configurar Ambiente React Native

Instale as dependências necessárias para desenvolvimento mobile.

npm install -g expo-cli
2

Criar Novo Projeto React Native

Inicie um novo projeto com a estrutura correta.

npx create-expo-app HelpNowMobile
cd HelpNowMobile
3

Instalar Dependências Necessárias

Adicione as bibliotecas específicas para React Native.

npm install @react-navigation/native
npm install react-native-screens react-native-safe-area-context
npm install @react-navigation/bottom-tabs
4

Migrar Código Fonte

Adapte seus componentes React para React Native.

# Substitua div por View
# Substitua button por TouchableOpacity
# Substitua input por TextInput
5

Testar em Dispositivos

Execute o aplicativo em emuladores e dispositivos reais.

npx expo start

Dependências Essenciais

Navegação

React Navigation
npm install @react-navigation/native
Navegação por Abas
npm install @react-navigation/bottom-tabs
Navegação em Stack
npm install @react-navigation/stack

UI Components

React Native Paper
npm install react-native-paper
Vector Icons
npm install react-native-vector-icons

Funcionalidades

Geolocalização
npm install react-native-geolocation-service
Mapas
npm install react-native-maps
Notificações
npm install react-native-push-notification
Nota: Algumas dependências necessitam de configuração nativa adicional. Consulte a documentação de cada pacote.

Configuração do Ambiente

Windows
macOS
Linux

Configuração para Windows

  1. Instale o Chocolatey (gerenciador de pacotes)
    Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))
  2. Instale o Node.js e Python
    choco install -y nodejs-lts python2 openjdk11
  3. Instale o Android Studio
  4. Configure as variáveis de ambiente ANDROID_HOME

Configuração para macOS

  1. Instale o Homebrew
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  2. Instale o Node.js
    brew install node
  3. Instale o Watchman
    brew install watchman
  4. Instale o Xcode para compilar para iOS

Configuração para Linux

  1. Instale o Node.js
    curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
    sudo apt-get install -y nodejs
  2. Instale o JDK
    sudo apt-get install -y openjdk-11-jdk
  3. Instale o Android Studio
  4. Configure as variáveis de ambiente
Atenção: A configuração do ambiente React Native pode ser complexa. Considere usar o Expo para simplificar o processo inicial.

Checklist de Migração

  • Configurar ambiente de desenvolvimento React Native
  • Criar novo projeto com expo ou React Native CLI
  • Instalar e configurar React Navigation
  • Substituir componentes HTML por componentes React Native
  • Adaptar estilos CSS para StyleSheet do React Native
  • Configurar permissões para recursos nativos (câmera, GPS, etc)
  • Testar em emulador Android
  • Testar em emulador iOS (se possível)
  • Testar em dispositivo físico
  • Configurar build para produção
Dica: Comece migrando uma tela por vez para facilitar o processo de debug e testes.
